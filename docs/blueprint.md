# **App Name**: BizSpark AI

## Core Features:

- Business Profile Setup: Multi-step form for users to input business name, category, description, upload logo, and specify products/services.
- AI-Powered Website Generation: Automatically generate a responsive, CMS-based website prototype using the provided business details and selected template.
- Website Deployment & Customization: Enable users to preview and publish their AI-generated website, connect a custom domain, and perform basic edits to website sections.
- Social Media Account Linking: Allow users to securely connect their Instagram, Facebook, TikTok, and X (formerly Twitter) accounts to the platform.
- AI Content Generation Tool: An AI tool that automatically converts new product announcements or business updates into tailored social media captions and website updates.
- Content Scheduling & Publishing: Provide a mechanism to review, schedule, and publish the AI-generated content to selected social media accounts and update the website.
- Unified Dashboard: Display a centralized dashboard showing website status, connected social media accounts, and recent AI-generated post activity.

## Style Guidelines:

- Primary brand color: A deep, authoritative yet vibrant violet (#6633CC). This reflects a blend of creativity and reliability, suitable for a tech-driven business platform.
- Background color: A very light, desaturated lavender-grey (#F1F0F5), providing a clean and understated canvas that allows content and interactive elements to stand out.
- Accent color: A bright, clear blue (#677CE4). This hue is used to draw attention to call-to-action buttons, key links, and highlight important information without overwhelming the user.
- Headline and body font: 'Inter' (sans-serif), chosen for its clean, modern, and highly legible appearance across various screen sizes, aligning with a professional SaaS aesthetic.
- Utilize a consistent set of minimalist, outline-style icons with subtly rounded corners to complement the clean interface and rounded card elements, enhancing visual harmony.
- Implement a 'Modern SaaS' inspired layout featuring sidebar navigation, clear content segmentation using rounded cards for data display, and ensuring responsiveness across all device types.
- Incorporate subtle and functional animations, such as smooth transitions for content loading, form submissions, and status updates, to enhance user feedback and overall experience without distractions.
